# Решение квадратных уравнений

Программа для решения квадратных уравнений вида ax² + bx + c = 0.

## Использование

### В браузере
Откройте файл `index.html` в веб-браузере.

### В консоли Node.js
```javascript
const solveEquation = require('./solveEquation');
console.log(solveEquation(1, -3, 2)); // [2, 1]
